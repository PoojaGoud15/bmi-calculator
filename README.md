# 🏥 Subhadra Hospitals – BMI Calculator

A responsive Body Mass Index (BMI) Calculator developed using **HTML, CSS, and JavaScript**. This application calculates BMI based on user input, categorizes health status, and provides personalized recommendations through a clean, hospital-inspired interface.

## ✨ Features

- Responsive hospital-themed user interface
- BMI calculation using height and weight
- Personalized health recommendations
- Color-coded BMI status indicators
- Input validation
- Reset functionality
- Interactive and user-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
